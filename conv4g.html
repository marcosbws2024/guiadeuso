1 Selecionar comandos disponíveis

Clique na imagem para expandir

Na tela principal, você encontra todos os comandos disponíveis para configuração dos dispositivos E3+4G. O gerador é muito parecido com o E3+2G, mas possui comandos exclusivos como Sensor de Jammer e Antifurto.

Dica: Selecione os checkboxes dos comandos que deseja configurar conforme a necessidade do cliente.

2 Configurar opções dos comandos

Clique na imagem para expandir

Após selecionar os comandos desejados, você pode configurar as opções de cada comando. Os campos disponíveis variam conforme o comando selecionado.

Sensor de Jammer

Detecta interferência e bloqueio de sinal GPS/GSM

Antifurto

Proteção avançada contra furto com alertas em tempo real

Observação: Cada comando possui opções específicas que devem ser configuradas conforme a necessidade do cliente.

4 Gerar Comandos

Clique na imagem para expandir

Após configurar todos os comandos desejados, clique no botão Gerar Comandos. O sistema irá:

  • Processar todas as configurações selecionadas
  • Gerar automaticamente os comandos SMS prontos para envio
  • Converter os comandos para múltiplos comandos por linha quando necessário
  • Exibir os comandos formatados para copiar e enviar
Exemplo de comando gerado: AT+APN=claro.com.br
Conversão automática: O sistema converte os comandos para múltiplos comandos por linha, otimizando o envio via SMS.
