1 Visão Geral do Conversor

Clique na imagem para expandir

O Conversor de Comandos serve para pegar comandos prontos e colocá-los no formato desejado:

  • Linha a Linha → cada comando em uma linha separada
  • Múltiplos Comandos por Linha → vários comandos em uma única linha, separados por ;
Dica: O conversor também calcula a conversão de horas e minutos em segundos para configurar HB e SHB, além de formatar CPF/CNPJ para consulta na Saitro.

2 Conversor 2G/4G e 4G/2G

Clique na imagem para expandir

O conversor permite alternar entre formatos de comando para diferentes tecnologias:

  • 2G/4G: Converte comandos do formato 2G para 4G
  • 4G/2G: Converte comandos do formato 4G para 2G
Utilidade: Facilita a adaptação de comandos entre diferentes modelos de equipamentos e tecnologias de rede.

3 Conversor de Horas/Minutos e Validador CPF/CNPJ

Clique na imagem para expandir

O conversor oferece duas funcionalidades essenciais:

Conversor de Horas e Minutos

Converte horas e minutos em segundos para configurar HB (Heartbeat) e SHB (Sleep Heartbeat) dos equipamentos.

1h 30min = 5.400 segundos
Validador de CPF e CNPJ

Valida e formata CPF/CNPJ para consulta correta na plataforma Saitro.

12345678000199 → 12.345.678/0001-99

4 Linha a Linha para Múltiplos Comandos

Clique na imagem para expandir

Converta comandos do formato Linha a Linha para Múltiplos Comandos por Linha:

Antes (Linha a Linha):
AT+APN=claro.com.br
AT+PORT=8080
AT+DNS=8.8.8.8
Depois (Múltiplos Comandos):
AT+APN=claro.com.br;+PORT=8080;+DNS=8.8.8.8
Dica: Múltiplos comandos por linha economizam créditos de SMS e agilizam a configuração remota.

5 Múltiplos Comandos para Linha a Linha

Clique na imagem para expandir

Converta comandos do formato Múltiplos Comandos por Linha para Linha a Linha:

Antes (Múltiplos Comandos):
AT+APN=claro.com.br;+PORT=8080;+DNS=8.8.8.8
Depois (Linha a Linha):
AT+APN=claro.com.br
AT+PORT=8080
AT+DNS=8.8.8.8
Observação: A conversão para Linha a Linha facilita a leitura e verificação de cada comando individualmente.

6 Formatação de CPF/CNPJ

Clique na imagem para expandir

O conversor formata CPF e CNPJ para consulta correta na plataforma Saitro:

CPF

Antes: 12345678901

Depois: 123.456.789-01

CNPJ

Antes: 12345678000199

Depois: 12.345.678/0001-99

Formatação correta: CPF e CNPJ formatados corretamente garantem consultas precisas na plataforma Saitro.
