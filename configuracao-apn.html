Configuração de APN

Procedimentos para estabelecer conexão GPRS em dispositivos de rastreamento.

Verificação Inicial

O chip deve estar ativo, com plano de dados M2M e inserido corretamente no slot antes do envio dos comandos SMS.

Parâmetros por Operadora

Operadora APN Usuário/Senha
Vivo smart.m2m.vivo.com.br vivo / vivo
Claro m2m.claro.com.br claro / claro
Arqia m2m.arqia.com.br arqia / arqia
Protocolo GT06

Envie para o número do chip no rastreador:

APN,123456,smart.m2m.vivo.com.br,vivo,vivo#
Protocolo Suntech

Exemplo para ST310U/ST340:

ST300CONF;01;smart.m2m.vivo.com.br;vivo;vivo;...

Dicas de Diagnóstico

  • Após o envio, aguarde a resposta "APN OK" ou similar.

  • Verifique o sinal GPRS usando o comando de status (ex: STATUS# ou PARAM#).
