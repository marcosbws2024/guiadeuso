Configuração de APN

Procedimentos para estabelecer conexão GPRS em dispositivos de rastreamento.

Verificação Inicial

O chip deve estar ativo e com plano de dados M2M antes do envio dos comandos.

Parâmetros por Operadora

Operadora APN Usuário/Senha
Vivo smart.m2m.vivo.com.br vivo / vivo
Claro m2m.claro.com.br claro / claro
Protocolo GT06
APN,123456,smart.m2m.vivo.com.br,vivo,vivo#
Protocolo Suntech
ST300CONF;01;smart.m2m.vivo.com.br;vivo;vivo;...

Diagnóstico

  • Aguarde a resposta SMS para confirmar a gravação (ex: APN OK).

  • Verifique se o LED de dados do rastreador mudou o padrão de piscada após a configuração.
