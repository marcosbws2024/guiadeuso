1 Visão Geral da Página

Clique na imagem para expandir

A Página de Downloads contém arquivos que podem auxiliar o atendimento, como manuais, guias e documentos técnicos. Cada arquivo possui dois botões de ação:

Botão Link

Abre uma nova página com o site de download do arquivo

Botão Ação

Copia o link para enviar diretamente ao cliente

Dica: Utilize o botão de ação para copiar o link e compartilhar rapidamente com o cliente via WhatsApp, e-mail ou SMS.

2 Buscar e Copiar Links

Clique na imagem para expandir

É possível buscar o arquivo desejado e copiar o link para enviar ao cliente. O processo é simples e rápido:

  • Digite o nome do arquivo ou parte dele no campo de busca
  • Localize o arquivo desejado na lista de resultados
  • Clique no botão de ação (copiar) para copiar o link
  • Cole o link no chat, e-mail ou SMS para enviar ao cliente
Copiado! O link é copiado automaticamente para a área de transferência, pronto para ser colado e enviado.

3 Abrir Link para Download

Clique na imagem para expandir

Ao clicar no botão de link, uma nova página será aberta com o site de download do arquivo. Isso permite que você:

  • Visualize o arquivo antes de compartilhar
  • Baixe o arquivo para seu computador
  • Verifique se o link está funcionando corretamente
Observação: O link abre em uma nova aba, mantendo a página de downloads aberta para continuar o atendimento.

4 Copiar Link sem Abrir

Quando clicamos no botão de ação, apenas copiamos o link sem abrir a página. Isso é útil para:

WhatsApp

Envie o link diretamente no chat do cliente

E-mail

Cole o link no corpo do e-mail

SMS

Envie o link via mensagem de texto

Exemplo de link copiado:
https://www.mediafire.com/file/9bzt1q72mpbjvid/Manual_E3%252BLONGLIFE_V.1.0.pdf/file
Dica: Copiar o link sem abrir a página é mais rápido e eficiente para o atendimento, permitindo enviar vários arquivos rapidamente.

6 Fluxo de Atendimento

Exemplo prático de como utilizar a página de downloads durante o atendimento:

  1. Cliente solicita um manual → O atendente acessa a página de downloads
  2. Busca pelo manual → Digita o nome do arquivo no campo de busca
  3. Localiza o arquivo → Encontra o manual desejado na lista
  4. Copia o link → Clica no botão de ação para copiar o link
  5. Envia ao cliente → Cola o link no WhatsApp, e-mail ou SMS
  6. Cliente acessa → O cliente clica no link e faz o download
